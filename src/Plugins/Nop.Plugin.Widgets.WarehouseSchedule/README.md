# Documentation: WarehouseSchedule Plugin

In this file we will be viewing the corresponding information about the WarehouseSchedule Plugin, this plugin is developed for NopCommerce v4.30
Its main functionality is to create schedules for a warehouse and to be able to manage whether it is going to be active or not and if it is not active, all working hours will be ignored.

# Compilation and installation

To configure the plugin you need to mount it in a NopCommerce corresponding to the version of the plugin, 4.30 in this case, and therefore we will need IIS and SQL Server
To run the project from Visual Studio, once this is done we can install the plugin from the administration by going to Configuration / Local Plugins and the Plugin should appear
from WarehouseSchedule Plugin, once installed, proceed to create a warehouse normally and the warehouse screen will appear modified to the corresponding needs.

# Fields and values

Several more fields were added to said screen.

> *There are the following fields:*

1. is Active: Indicates whether the day is active or not.

2. Day: on which the warehouse will be open.

3. Begin time: when the warehouse opens on that day.

4. End time: when the warehouse closes on that day.
