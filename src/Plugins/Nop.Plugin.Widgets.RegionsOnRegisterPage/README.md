# Documentation: RegionsOnRegisterPage plugin

<div style="text-align: justify">

In this file we will be viewing the corresponding information about the RegionsOnRegisterPage plugin, this plugin is developed for NopCommerce v4.30. Its operation is described as follows: after having previously installed the plugin, a dropdown box will appear in the register and the customer info page, where we can proceed to choose a region.

# Compilation and installation

To configure the plugin, you need to add the project in the NopCommerce solution with which we are going to work, corresponding to the plugin version, **4.30** in this case, and, therefore, we will need IIS and SQL Server to run the project from Visual Studio. Once this is done, we can install the plugin from the administration, entering Configuration / Local Plugins. Next, rememeber to add all the regions in the database, in the table named **"Region"**. 

If the previous steps were fulfilled, the RegionsOnRegisterPage Plugin will appear in the register and customer info page. Once installed, the customer can proceed to regsiter choosing the region, and edit the region selected in the customer info.
